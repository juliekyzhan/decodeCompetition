# Replication Materials

This repository contains data and code for the paper:
"Decoding Asymmetric Decision Making: U.S. Congressional Legislative Behavior on China"

## Project Status
This project was not preregistered prior to data analysis.

## Structure
- /data: datasets used in the analysis
- /code: scripts for descriptive analysis, models, and robustness checks
- /output: tables and figures

## Reproducibility
Run scripts in the /code directory in the order specified to reproduce results.
